Il programma è stato scritto con il linguaggio di programmazione OO Java sulla piattaforma Eclipse Luna 4.4.
Per poter aprire il progetto ed eseguire correttamente il progetto bisogna seguire queste istruzioni:
- Aprire Eclipse;
- Selezionare in alto a sinistra File -> Import;
- Una volta aperta la finestra di import selezionare dal tab General "Existing Projects into Workspace" e cliccare su 
  "Next";
- Nella schermata successiva cliccare su "Select root directory" e successivamente su "Browse.." per scegliere la 
  directory in cui è contenuto il progetto "Progetto_Linguaggi_ParserLL1";
- Una volta selezionato il progetto da importare cliccare su "Finish".
Per realizzare le tabelle di parsing è stata usata la libreria Batik 1.8 che permette la definizione di file in formato
SVG. Per integrare la libreria è necessario seguire le seguenti indicazioni:
- Selezionare il progetto contenuto nel "Package Explorer" di Eclipse;
- Cliccare su Project -> Properties nel menù in alto oppure cliccare sul progetto con il tasto destro e selezionare
  direttamente Properties;
- Selezionare "Java Build Path" a sinistra;
- Andare sul tab "Libraries";
- Cliccare a destra su "Add External JARs.." e scegliere la directory che conduce alla cartella "Batik 1.8";
- Aprire la cartella e scegliere tra i file .jar disponibili "batik-1.8";
- Cliccare su "OK".
Una volta eseguite queste indicazioni è possibile mandare correttamente in run il progetto.
